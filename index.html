Step 01 — Mode

作業のモードを選んでください。既存アプリの改修・新旧仕様移行にも対応します。

仕様移行

Step 02 — Brief

要件を入力

作りたいアプリの目的と機能を書いてください。詳しいほど設計図の精度が上がります。

🏠 ハーツハウス標準仕様を注入

ES5(var) / fbReady パターン / askAI エンドポイント + x-access-key / extractGeminiText ヘルパー / 既知の禁止事項(useSearch+responseSchema併用禁止 等) を自動でプロンプトに含めます。

🔐 管理者認証テンプレートを注入

ログインゲート(新規登録なし) / 共通 users コレクション参照 / isAdmin 判定の管理パネル / セカンダリインスタンスでの仮パスワード発行 を生成コードに組み込みます。日報・チャット等と同じアカウントが使い回せます。
